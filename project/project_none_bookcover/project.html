그리고 아무도 없었다 북커버 디자인

까마귀들이 불길한 상황을 연출 할때 많이 활용되는 까마귀들을 배치하여 더욱 추리소설만의 공포스러운 분위기를 연출하고자 하였다. 뒷표지에는 문장을 나누는 선을 넣어서 가독성을 높이고자 하였 '추리 소설의 요왕 아가사 크리스티 장편소설' 부분에 마치 피가 지나간 것 같이 그어 강조를 하고자 하였다. 책 띠지에 색상을 블루톤을 사용하여 공포스러운 분위기를 연출하였다. 해문 출판사 심볼 로그는 월식을 모티브로 제작하였다. 책 제목에 까마귀를 앉혀 놓아 제목을 더욱 일관성 있게 하였다. 폰트는 목각파임 M, HY견명조를 사용하였다.
